template<class GT >
void afg::import_asconnect ( GT &  gra,
std::istream &  is 
)

import graph from an "asconnect" file. Asconnect file is a list of AS's and their interconnections, available for download from: http://www.antc.uoregon.edu/route-views/.

File format per line: as_index -> #_of_neighbors : a list of AS neighbors separated by ":". The imported graph will have vertex data of int type (AS index), each edge will have weight 1(type int). All links are assumed to be bi-directional.

import graph from the alt file by GeorgiaTech graph generator. File format (transit stub graph): first line is some explaination
2nd line: number_of_nodes number_of_edges keyword"transtub"
3rd line: empty line
4th line: vertex line
list of nodes (as many as number_nodes)
empty line
edge line
list of links (as many as half of number_edges -- bidirectional links)
GT is the graph type, it must have vertex data type tsVertex and edge data type int (or convertions from them provided).
