graph_type< VDT, CEdgeW2< WT1, double > > &  ngra

graph of single weight -> two double weights. In new graph, each edge has two weights: one is the original, one is a random number between 0 and 1. ngra should be cleared fist (or newly created without inserting any vertex) before passed to this function. VT is the type of vertex data and WT1 is the type of original weight.

gra0original graph
ngranew graph
