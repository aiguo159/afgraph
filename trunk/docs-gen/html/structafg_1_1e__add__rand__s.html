Detailed Description

template<class EDT1, class DT>
struct afg::e_add_rand_s< EDT1, DT >

add symmetrical random number as a new edge weight. New edge has two weights: the original one and a random number. New weight is symmetrical: w(i->j)=w(j->i) EDT1: original edge type; DT: random number type (e.g., double or int).
