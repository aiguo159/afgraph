Detailed Description

A Framework for Graph (network) data structures and algorithms.

This file provides a number of type definitions to save some typing time. Before including any file that uses types defined in this file, make sure the header file(s) in which graph_type and/or rtree_type are defined is(are) included first. For example, graph.h provides graph_type and rtree.h provides rtree_type. This way, types defined here and other functions/clsses that use these tyepes will not be tied to a specific graph or rtree type.
