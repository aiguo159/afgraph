void afl::rand::init ( int  n = 111193) [inline]

Initialize the random seed with system time. Use system time divided by a specific integer (default is 111193) as the seed. Thus makes it possible for a program to initialize the seed with a different number every time it runs though the integer passed to init_rand() is the same.

Returns:
none
