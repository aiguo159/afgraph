Dijkstra algorithm to compute single source shortest paths. GraphT: type of the graph.

Parameters:
graphgraph on which to run the algorithm, no "negative" weighted edge. Graph is not required to be "packed".
nsourcesource node, should be between 0 and graph.szie()-1
f_weightfunction object to get weight of an edge (passed by pointer), e.g.: pointer2value<int>( ), runit_p<..>( ).
w_inftyan "upper bound" value of weight which should be larger than the path length of any valid path; used to initialize distance to all nodes.
preda vector to store predecessor information, must have size()>=graph.range().
dista vector to store path length information (from nsource to a node), must have size()>=graph.range(); if there is no path from nsource to node i, then dist[i]=w_infty.
Exceptions:
unknown_error
